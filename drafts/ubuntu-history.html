Ubuntu history

原文：http://dl.fullcirclemagazine.org/issue0_en.pdf

2004年10月, 4.10 － 长疣的疣猪

长疣的疣猪是Ubuntu发布的第一个版本，之所以这样叫它是因为它具有毫不掩饰缺陷（Warts and all）的特点。为什么要从4开始呢？ 4代表了它发布的年份；2004，更准确的说是2004年10月份。它的桌面颜色有些暗淡，橙色和棕色象征了南非的部落。值得一提的是Ubuntu采用Live CD 的方式来启动。Live CD 能够允许潜在的用户在没有安装的时候就可以使用它。这是其他的Linux发行版并没有提供给用户的功能。

  • Linux Kernel - 2.6.7, xfree86 - 4.3.0.1, Gnome - 2.8

2005年四月, 5.04 － 灰白的刺猬

6个月后，作为承若，灰白的刺猬到来了。桌面的颜色变鲜明了一些，但对比现在的标准仍然比较暗淡。 Live CD 扩展到包含64位系统和Power PC。在2005年4月，Ubuntu已经成熟并发布了一个新的版本叫作Kubuntu。它是建立在一个叫作KED的桌面环境系统之上，来代替了GNOME。

  • Linux kernel - 2.6.10, X.Org - 6.8.2, Gnome - 2.10, KDE - 3.4

2005年10月, 5.10 － 活泼的獾

Ubuntu 是现在最受社区欢迎的版本。伴随着Breezy的到来，Edubuntu诞生了，这第三个版本目标瞄准了教育组织领域。这些衍生版本也有了对服务器的支持。它很好强（Not to be outdone），Kubuntu 发行版用Adept替换了Kynaptic（一个包管理器），它成为了第一个使用devtags来简化了在Adept中搜索应用程序。KDE也添加了对蓝牙的支持。

  • Linux kernel - 2.6.12, X.Org - 6.8.2, Gnome - 2.12.1, KDE - 3.5.2

2006年6月, 6.06 － 整洁的公鸭

Dapper 成为了第一个被没有遵守半年承若的跳票版本。4月来了又走，伴随而来没有发布的原因是Mark Shuttleworth（Ubuntu 的创始人）追加了开发的时间。这也是第一次增加了LTS－长期支持 的Ubuntu版本。桌面系统版本有3年的支持，服务器的有5年。另一个第一次是；Live CD 也支持成了安装光盘。在Live CD（现在是叫作Desktop CD）里用户能点击图标轻松的去安装Ubuntu。不仅用户能安装Ubuntu 到他们的硬盘里，随着Dapper用户还能安装Ubuntu到USB设备里。其他的扩展是，内核包含了更快的启动时间，关机界面化，一个新的升级工具，更好的视频回放。网络依然很复杂，但新的网络管理工具使它简单了一些。安装程序里默认的OpenOffice 升级到了2.0.2版本，如果这些还不够，Ubuntu家族又一次添加了新的成员。除了使用 Gnome 或者 KDE 之外，Xubuntu 使用了 Xfce桌面环境。较少的占用资源，Xubuntu能比较完美的运行于老的台式机和笔记本之上。

  • Linux kernel - 2.6.15, X.Org - 7.0, Gnome - 2.14, KDE 3.5.4, Xfce 4.3.9 0.1

2006年10月, 6.10 － 急躁的水蜥

伴随着更快的启动、关机速度 和集成了TomBoy、F-Spot，Ubuntu Edgy 成为了在Dapper和即将发行的Feisty Fawn 7.04间的一个小小的里程碑。Ubuntu的界面也变的明亮悦目起来。Kubuntu添加了图片管理软件，digiKam；重新设计了系统的设置面板；并且改良了笔记本按键 和 电源管理支持。

  • Linux kernel - 2.6.17, X.Org - 7.1, Gnome - 2.16, KDE - 3.5.5, Xfce - 4.3.9 9 RC1

2007年4月, 7.04 － 坏脾气的小鹿

最新（本杂志发布时）发布的Ubuntu包含了更多的新特性，其中包含了更容易的方式去安装多媒体解码器和二进制驱动，而且网络也改善了，当然还有Windows迁移助手等等。往下读吧 ！

  • Linux kernel - 2.6.20, X.Org - 7.2, Gnome - 2.18, KDE - 3.5.6, Xfce 4.4
